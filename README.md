InventoryManagementSystem
🚀 AI-Powered Inventory Management System (IMS)

A modern and intelligent inventory management system designed for small and medium-sized businesses. This project leverages the power of Artificial Intelligence to optimize stock prediction and automate manual processes.

👥 Team and Roles

The project is developed by a team of three members following an Agile methodology:

Team Leader (TL): Responsible for Backend Architecture, Database Management (Supabase), AI Integration, and Code Reviews.
Developer 1 (Dev1): Focused on Business Logic, Stock APIs, Authentication, and Security.
Developer 2 (Dev2): Handles UI/UX Design (Tailwind), State Management (TanStack Query), and Visual Reporting.
🛠️ Technology Stack
Framework: Next.js (App Router)
Database & Authentication: Supabase (PostgreSQL)
Data Fetching: TanStack Query v5
Styling: Tailwind CSS
AI Integration: Vercel AI SDK / OpenAI API
Data Visualization: Recharts
Invoicing: jsPDF
✨ Core Features
🔹 Inventory Management
Smart Dashboard: Provides a real-time overview of sales performance and inventory value.
Low Stock Alerts: Automatically notifies users when products reach critical stock levels.
Categorization: Enables advanced organization of products by type and supplier.
🤖 AI Integration (Key Market Differentiator)
AI Stock Predictor: Analyzes historical sales data to forecast future supply requirements.
AI Smart Scanner: Automatically populates product fields through invoice scanning (OCR).
AI SEO Descriptions: Generates product descriptions optimized for marketing purposes.
💳 Sales and Reporting
POS Interface: A fast and efficient interface for point-of-sale operations.
Invoice Generation: Instantly generates professional PDF invoices after each transaction.
Data Export: Supports exporting reports in CSV/Excel formats.
